AXIS-SOW-POC
============

Installation
------------

1. Install nodejs
    Use this link https://nodejs.org/en/ or a packet manager
2. Clone the GIT
3. Open a terminal
4. Move to the project folder
5. Run npm install
    This will install/update all the dependencies
6. Run npm start
    This will launch the project
7. Open a browser and go to localhost:3000/
